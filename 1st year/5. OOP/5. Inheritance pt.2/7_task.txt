Напишите базовый класс Board для создания полей для различных игр. Принимает параметры: количество строк, количество столбцов, заполнитель, по умолчанию 0.

При инициализации создаёт двумерный список такого размера и с таким заполнением. Может выводить список построчно без разделителей ( __str__).

Также напишите его наследник, создающий квадратное поле для игры в шашки (Checker). Принимает только размер одной стороны и два заполнителя: обозначение пустых черных клеток (нижняя левая, по умолчанию 0) и пустых белых клеток (вторая снизу левая), клетки чередуются.
Также имеет метод расставить шашки (put_on); принимает аргументы – обозначение черных и белых шашек; белые ставятся на 3 ряда снизу, черные – на 3 ряда сверху.

От шашечного поля унаследуйте класс, создающий поле для шахмат (Chess) размером 8 на 8, заполнитель белых клеток 1, черных 0. Инициализируется без аргументов. Отличается только метод расстановки фигур, который вызывается также с аргументами – функциями для преобразования обозначений черных и белых фигур, по умолчанию черные обозначаются строчными буквами, белые прописными. Обозначения фигур приведены ниже:
P – пешка;
R – ладья;
N – конь;
B – слон;
Q – ферзь;
K – король.
