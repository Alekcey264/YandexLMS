Из того ли то из города из Мурома,
Из того села да Карачарова
Выезжал удаленький дородный добрый молодец.

Ну вот насколько он удаленький по сравнению с другими богатырями сейчас и посмотрим.

Напишите класс EpicHero (Богатырь), экземпляр которого создается с атрибутами:
имя;
количество побед;
список оружия, которым владеет богатырь.

У класса есть методы:
 __str__ – возвращает строку в формате Epic Hero <имя>, <количество побед>;
 __repr__ – возвращает представление в виде EpicHero(‘<name>’, <number_of_wins>, <list_of_weapons>), список должен быть отсортирован;
add_win() – добавляет победу;
add_weapon(item) – добавляет оружие;
del_weapon(item) – удаляет оружие, если оно есть, если нет, ничего не делает;
поддерживаются все операции сравнения, сравниваются в порядке: по количеству побед, по количеству видов оружия, по длине имени по убыванию (чем короче имя, тем “больше” богатырь), по имени по алфавиту.
