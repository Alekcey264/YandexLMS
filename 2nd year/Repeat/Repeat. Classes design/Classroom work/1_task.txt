Добавим в класс Note сравнение нот. Сравнивать будем по высоте, без учёта длительности, причём будем считать большей ту ноту, чей тон выше (частота колебаний больше).

Также добавим возможность переноса звука выше или ниже. У нас получится упрощённая версия транспонирования, так как на нашем ксилофоне есть только «белые» клавиши. Перенос будет происходить с помощью сдвигов >> вверх и << вниз на гарантированно целое положительное значение. Например, сдвиг «до» на три вправо (>>3) приводит к «фа».

Сдвиг должен работать циклически: если сдвинуть ту же «до» на один вниз (<<1), должна получиться «си». А если сдвинуть «си» на один выше (>>1) — опять «до».

И ещё один новый метод — get_interval, позволяющий получить значение интервала между двумя нотами: «прима», «секунда», «терция», «кварта», «квинта», «ceкcта» или «септима», где «прима» — интервал между двумя равными по тону нотами, а «септима» — максимальный доступный на нашем ксилофоне интервал — между «до» и «си».

В решении нужно объявить и можно использовать следующие константы (тестирующая система будет импортировать их из вашего кода):

N = 7  
PITCHES = ["до", "ре", "ми", "фа", "соль", "ля", "си"]  
LONG_PITCHES = ["до-о", "ре-э", "ми-и", "фа-а", "со-оль", "ля-а", "си-и"]  
INTERVALS = ["прима", "секунда", "терция", "кварта", "квинта", "секста", "септима"]
