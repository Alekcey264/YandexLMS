Для тренировки работы в командной строке выполните следующие действия:

Посмотрите документацию к команде mkdir и одной командой в терминале рекурсивно создайте в текущей папке следующую структуру "/tmp/.tasks/bash/one/"
Перейдите в директорию "/tmp/" и выведите списком ее содержимое с размерами (размеры должны быть в привычном для человека формате - мегабайты, гигабайты и т.д.)
Переместите созданную папку ".tasks/bash/one" в ".tasks/one" и удалите ".tasks/bash"
Формат вывода
В качестве ответа укажите последовательность команд, которая приводит к решению задачи

Примечания
В качестве ответа пришлите набор команд, который по Вашему мнению необходим для решения задачи.
Python здесь не требуется, а задача будет проверена преподавателем.
