С помощью библиотеки unittest напишите тесты для функции reverse(s) из предыдущего урока. Напомним, что функция reverse(s) должна возвращать строку, состоящую из символов входной строки, записанных в обратном порядке. Функцию reverse(s) нужно импортировать из модуля yandex_testing_lesson.

Проверьте следующие входные данные:

Пустая строка
Строка из одного символа
Строка-палиндром из нескольких символов
Обычная (т.е. не палиндром) строка из нескольких символов
Не строка, не итерируемый объект (например, int)
Не строка, итерируемый объект (например, list)
Перед отправкой решения запустите тесты на заведомо неправильной реализации функции reverse(s). Посмотрите на сообщения о не пройденных тестах. Легко ли понять, что именно сломалось?
