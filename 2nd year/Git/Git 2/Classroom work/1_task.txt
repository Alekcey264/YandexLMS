Последовательно выполните следующее задания:

Создайте в PyCharm проект git_lab1_lesson2, откройте его
Склонируйте в него репозиторий https://github.com/YandexLyceum/human.git
Изучите список всех веток репозитория
Просмотрите содержимое файла human.txt
Сравните содержимое файла human.txt в ветке master с каждой из веток, которые вы видите
Создайте ветку boots_buttons от ветки master, переключитесь в неё
Слейте в новую ветку ветки boots и buttons
Залейте ветку boots_buttons (но не все остальные ветки) в свой собственный репозиторий на GitHub и в качестве решения этой задачи пришлите ссылку на него
