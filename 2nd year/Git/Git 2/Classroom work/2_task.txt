Разбейтесь на группы по 2-3 человека, выберите 1 человека, который создаст репозиторий (назовем его тим-лид — лидер команды).
Тим-лиду необходимо создать на GitHub репозиторий с именем gitflow и дать доступ к репозиторию остальным участникам группы, для этого необходимо перейти в настройки репозитория (меню "Settings", раздел "Collaborators)" по ссылке , где «LiceumStudent001» необходимо заменить на GitHub-логин тим-лида. И в этом разделе добавить ("Add colaborator") всех участников своей группы (2-3 человека), найдя их по логинам GitHub.
Каждому участнику группы необходимо принять приглашение в репозиторий, перейдя по ссылке, где «LiceumStudent001» необходимо заменить на GitHub логин тим-лида и нажав «Accept invitation».
Если пункты 1-3 выполнены корректно, то у всех участников группы появится доступ на запись к созданному тим-лидом репозиторию.
Далее всем участникам группы нужно на своем компьютере в своей домашней директории создать папку git_lab2_lesson2, перейти в неё и склонировать пустой репозиторий, созданный тим-лидом.
Отлично, на этом шаге у каждого участника группы должна быть директория git_lab2_lesson2 с пустым локальным репозиторием и единым удаленным репозиторием. Попробуйте поработать по gitflow, для этого пусть каждый реализует в отдельных ветках 3 фичи:

feature/userYourName (это — имя ветки), содержащую файл YourName.txt, в котором указан год вашего рождения;
feature/listYourName, содержащую файл list.txt с вашим ФИО и
feature/schoolYourName, содержащий файл YourNameSchool.txt, содержащий номер вашей школы.
Например, если вас зовут Вася, вы учитесь в школе №1523 и родились в 2004 году, то у вас должно получиться 3 фича-ветки:

Ветка	Файл	Содержимое
feature/userVasya	Vasya.txt	2004
feature/listVasya	list.txt	Vasya
feature/schoolVasya	VasyaSchool.txt	1523
 

Примечания
Создавать файлы можно в любом текстовом редакторе.
Задание проверяется преподавателем.
